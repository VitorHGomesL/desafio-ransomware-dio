Desafio Dio

Este projeto tem como objetivo implementar um Ransomware para criptografar arquivos utilizando a linguagem Python, conforme proposto no desafio da DIO da Formação em CyberSegurança.

O projeto difere do original pelo uso da biblioteca Cryptography do Python, mais utilizada no mercado do que a usada durante o desafio (pyaes), além de métodos mais eficientes de Criptografia para aprofundar os estudos no tópico.

Arquivos do projeto:

    encrypter.py: código que irá criptografar os arquivos;
    decrypter.py: código que irá descriptografar um arquivo previamente criptografado.
